# 日本社会人標準チェッカー 2026

タイムを入力すると、**第9回 日本社会人選手権水泳競技大会（2026）の参加標準記録**を突破しているか（突破／あと◯秒）を判定するWebアプリです。男女・種目・距離・タイムを選ぶだけで判定できます。

- 単一の `index.html` だけで動作（外部CSS/JS/画像・通信なし）。オフラインでもスマホでもそのまま開けます。
- スマホ最優先レイアウト（375px / 320px で横はみ出しゼロを確認済み）。

## 使い方

1. `index.html` をブラウザで開く（またはダブルクリック）。
2. **男女 / 種目 / 距離** を選び、**タイム**を入力（例：`55.14` または `2:01.13`）。
3. 標準記録に対して「突破」または「あと◯秒」が表示されます。

## 判定のルール

- **25m（短水路）・50m（長水路）どちらの公認記録**でも、同じ標準に対して判定します。
- 2025年4月1日以降の公式・公認競技会、または日本マスターズ協会公認競技会の記録が対象です。
- **同記録も突破扱い**（入力タイム ≦ 標準記録 で突破）。判定は 1/100 秒まで。

### 対応する種目・距離

- 自由形：50 / 100 / 200 / 400 / 800（女子）/ 1500m（男子）
- 背泳ぎ：50 / 100 / 200m
- 平泳ぎ：50 / 100 / 200m
- バタフライ：50 / 100 / 200m
- 個人メドレー：200 / 400m

> 自由形の 800m は女子のみ、1500m は男子のみ収録しています。

## 公開方法（GitHub Pages / Cloudflare Pages）

このフォルダは `index.html` がルートにある静的サイトなので、そのまま公開できます。

**GitHub Pages**
1. このフォルダをGitHubリポジトリにアップロード（`index.html` はリポジトリの直下に置く）。
2. リポジトリの Settings → Pages → Source を `main` ブランチ / `/ (root)` に設定。
3. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。

**Cloudflare Pages**
1. Cloudflare Pages で新規プロジェクトを作成し、このリポジトリを接続（またはフォルダを直接アップロード）。
2. ビルド設定は不要（フレームワークなし／ビルドコマンド空欄／出力ディレクトリはルート）。
3. デプロイ後、割り当てられた `*.pages.dev` のURLで公開されます。

> ビルド工程は不要です。`index.html` を1枚置くだけで動きます。

## 毎年の更新

標準記録が変わったら、`index.html` 内の `const ENTRIES = [ ... ]` の各 `standard` 値を差し替えてください。
